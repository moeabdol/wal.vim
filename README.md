# wal.vim
## Updated to support termguicolors

[![MIT licensed](https://img.shields.io/badge/license-MIT-blue.svg)](./LICENSE.md)

A vim colorscheme for use with **[pywal](https://github.com/dylanaraps/pywal)**.


## Installation

```vim
" Using plug
Plug 'typkkrft/wal.vim'

colorscheme gupywal
```
